# gpa-calculator
This program calculates your mark or gpa (current) given the marks in percentage

How to get started with development,
- Install Flask for Python 3

  MAC:
    Run this command on terminal
    - pip3 install flask
    - pip3 install flask-cachebuster
    
  WINDOWS:
    - pip install flask
